# **App Name**: GeminiSimplify

## Core Features:

- Input Area: Modern and responsive input area where users can paste text for simplification and translation.
- AI Simplification and Translation: Uses the Gemini API tool to simplify and translate the input text to the selected language.
- Output Display: Displays the simplified text in a clear, visually appealing output area.

## Style Guidelines:

- Primary color: Soft Lavender (#E6E6FA) for a calm and inviting feel.
- Secondary color: Light Peach (#FFE5B4) for background accents and highlights.
- Accent color: Deep Teal (#008080) for buttons and interactive elements to provide a modern and trustworthy feel.
- Use a fluid grid layout with plenty of whitespace for a modern and open feel.
- Incorporate vibrant, custom icons with a playful and modern aesthetic.
- Implement subtle, delightful animations for loading states and user interactions to enhance engagement.
